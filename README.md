todoApp-with-yeoman
===================

Todo app using AngularJS, Bower and Grunt during the Yeoman hackathon - Toronto

#Installing Dependencies:
  - Run ```npm install```
  - Run ```bower install```

#Running the app:
  - ```grunt serve```

#Optional:
  - To use Yeoman, run ```yo``` in your project directory and extend it with more generators/awesomeness!.
